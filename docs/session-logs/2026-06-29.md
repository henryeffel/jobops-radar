- `git diff --check` was run to detect whitespace errors.
- Repository search verified the workaround appears in both required documents.
- Application tests were not rerun because this task changes documentation and
  ignore rules only.

### Backend Concepts Used

No backend runtime concept was introduced. The workaround controls the operating
system location pytest uses for temporary test artifacts.

### CS / DSA Concepts Used

No DSA was used. The closest CS concept is filesystem permissions and process
environment inheritance: pytest reads `TMP` and `TEMP` from its process
environment when selecting a temporary directory.

### Design Decisions

A repository-local ignored directory avoids the inaccessible Windows system temp
path without changing application code or committing generated test artifacts.

### Issues / Warnings

- Observed error:
  `PermissionError: [WinError 5] Access is denied:
  C:\Users\Henry\AppData\Local\Temp\pytest-of-Henry`.
- PowerShell workaround:

  ```powershell
  mkdir .tmp_pytest
  $env:TMP = "$PWD\.tmp_pytest"
  $env:TEMP = "$PWD\.tmp_pytest"
  pytest
  ```

- The environment variable assignments apply to the current PowerShell session.
